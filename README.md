## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS

### Setup

    1. git https://github.com/ntzwilly/JS-capstone.git
    2. cd JS-capstone

### Install

    npm install

### Usage

    npm start
